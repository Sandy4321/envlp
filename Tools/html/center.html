center

Substract the mean of each column

Contents

  • Usage
  • Description
  • Example

Usage

XC = center(X)

Input

  • X: A matrix or a column vector.

Output

  • XC: A matrix or a column vector with the mean for each column equal to 0.

Description

This function centerizes a matrix or a vector, by substracting each column by its column mean.

Example

a = [1 2 3; 4 5 6];

center(a)

ans =

   -1.5000   -1.5000   -1.5000
    1.5000    1.5000    1.5000
